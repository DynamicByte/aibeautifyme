# Skincare Ingredient Interactions Guide

## Understanding Ingredient Interactions

Not all skincare ingredients play well together. Some combinations can reduce effectiveness, while others can cause irritation or even damage skin. This guide helps you understand what to combine and what to avoid.

## Quick Reference Chart

### Safe Combinations ✓

| Ingredient 1 | Ingredient 2 | Notes |
|--------------|--------------|-------|
| Hyaluronic Acid | Everything | Safe with all ingredients |
| Niacinamide | Retinol | Niacinamide reduces retinol irritation |
| Vitamin C | Vitamin E | Synergistic antioxidant protection |
| Vitamin C | Ferulic Acid | Stabilizes and enhances Vitamin C |
| Ceramides | Everything | Supports all other treatments |
| Peptides | Hyaluronic Acid | Complementary hydration and repair |
| SPF | Everything | Always use as final step |

### Use with Caution ⚠️

| Ingredient 1 | Ingredient 2 | Recommendation |
|--------------|--------------|----------------|
| Vitamin C | Niacinamide | May cause flushing; use separately |
| Vitamin C | Retinol | Use at different times of day |
| Retinol | AHA/BHA | Don't use same night |
| AHA | BHA | Can combine, but watch for irritation |
| Retinol | Benzoyl Peroxide | Use at different times |

### Never Combine ✗

| Ingredient 1 | Ingredient 2 | Why |
|--------------|--------------|-----|
| Retinol | Retinol | Never layer retinoids |
| Benzoyl Peroxide | Vitamin C | BP oxidizes Vitamin C |
| AHA/BHA | Retinol (same routine) | Over-exfoliation risk |
| Vitamin C | Copper Peptides | Copper oxidizes Vitamin C |

## Detailed Interaction Guide

### Retinol Interactions

**SAFE to combine with Retinol:**
- Hyaluronic Acid (excellent - counteracts dryness)
- Niacinamide (helps reduce irritation)
- Ceramides (supports barrier)
- Peptides (complementary anti-aging)
- Squalane (moisturizing buffer)

**AVOID with Retinol (same routine):**
- AHA (Glycolic, Lactic Acid)
- BHA (Salicylic Acid)
- Vitamin C (use at different times)
- Benzoyl Peroxide
- Other retinoids

**Best Practice:** Use retinol at night alone, acids and Vitamin C in AM or on alternate nights.

---

### Vitamin C Interactions

**SAFE to combine with Vitamin C:**
- Vitamin E (enhances efficacy)
- Ferulic Acid (stabilizes, doubles protection)
- Hyaluronic Acid (no interaction)
- SPF (C enhances sun protection)

**CAUTION with Vitamin C:**
- Niacinamide (older research suggests conflict; modern formulas usually fine, but may cause flushing)
- Retinol (different pH optima; use at separate times)
- AHA/BHA (increased irritation potential)

**AVOID with Vitamin C:**
- Benzoyl Peroxide (oxidizes Vitamin C)
- Copper Peptides (oxidation risk)

**Best Practice:** Use Vitamin C in the morning before SPF.

---

### AHA/BHA Interactions

**SAFE to combine:**
- Hyaluronic Acid (helps counteract dryness)
- Niacinamide (after waiting time)
- Ceramides (barrier support)

**CAUTION:**
- Other acids (don't over-exfoliate)
- Vitamin C (some formulas okay, watch for irritation)

**AVOID:**
- Retinol (same night)
- Benzoyl Peroxide (too harsh)
- Physical exfoliants (over-exfoliation)

**Best Practice:** Use acids 2-3x per week. On non-acid nights, use retinol.

---

### Niacinamide Interactions

**SAFE to combine with Niacinamide:**
- Almost everything!
- Retinol (actually helps reduce irritation)
- Hyaluronic Acid
- Ceramides
- Peptides
- AHA/BHA (after waiting period)

**CAUTION:**
- Vitamin C (controversial - may cause temporary flushing)
  - Modern research suggests it's fine
  - Some people are sensitive to the combination
  - If concerned, use at different times

**Best Practice:** Niacinamide is very versatile. Use AM and/or PM.

---

### Benzoyl Peroxide Interactions

**SAFE to combine:**
- Hyaluronic Acid
- Niacinamide
- Ceramides

**AVOID:**
- Retinol (can destabilize both)
- Vitamin C (oxidizes Vitamin C)
- AHA/BHA (over-irritation)

**Best Practice:** Use BP as spot treatment or in AM; other actives in PM.

## Building a Safe Routine

### Morning Routine Order
1. Cleanser
2. Toner (if using)
3. Vitamin C Serum
4. Niacinamide (or other water-based serums)
5. Hyaluronic Acid Moisturizer
6. Sunscreen (always last!)

### Evening Routine Order
1. Oil Cleanser (if wearing makeup/SPF)
2. Water-based Cleanser
3. Exfoliating Toner (2-3x week) OR skip on retinol nights
4. Retinol (on non-acid nights)
5. Niacinamide Serum
6. Eye Cream
7. Moisturizer

### Weekly Schedule Example

| Day | AM | PM |
|-----|-----|-----|
| Mon | Vitamin C + SPF | Retinol |
| Tue | Vitamin C + SPF | Niacinamide only |
| Wed | Vitamin C + SPF | AHA/BHA Toner |
| Thu | Vitamin C + SPF | Retinol |
| Fri | Vitamin C + SPF | Niacinamide only |
| Sat | Vitamin C + SPF | AHA/BHA Toner |
| Sun | Vitamin C + SPF | Retinol |

## Wait Times Between Products

Some ingredients need time to absorb or work at their optimal pH before applying the next step:

| Product | Wait Time |
|---------|-----------|
| Vitamin C (L-Ascorbic Acid) | 1-2 minutes |
| AHA/BHA Toner | 1 minute (or until dry) |
| Retinol | Apply to dry skin (wait 20-30 min after cleansing) |
| Regular serums | Until absorbed (30 seconds - 1 minute) |
| Moisturizer | Until absorbed |

## Signs You're Overdoing It

Watch for these signs of over-treatment:

1. **Persistent redness** that doesn't go away
2. **Burning or stinging** with normally gentle products
3. **Excessive dryness** or flaking
4. **Increased breakouts** (beyond normal purging)
5. **Skin feels "raw"** or sensitized
6. **Products that worked before** now irritate

### How to Fix Over-Exfoliated Skin

1. **Stop all actives** (retinol, acids, Vitamin C)
2. Use only gentle cleanser + moisturizer + SPF
3. Add barrier-repair ingredients (ceramides, centella)
4. Wait 1-2 weeks for skin to recover
5. Reintroduce actives slowly, one at a time

---

## Golden Rules

1. **When in doubt, use at different times** - AM vs PM, or alternate nights
2. **Introduce one new active at a time** - Wait 2 weeks between new products
3. **Listen to your skin** - If it's irritated, scale back
4. **More is not better** - You don't need every active every day
5. **SPF is non-negotiable** - Especially when using actives
6. **Hydration supports everything** - Hyaluronic acid and moisturizer help all treatments work better
